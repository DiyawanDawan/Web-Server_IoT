Post Data Sensor

URL: /v1/api/postData

Method: POST

Request Body:

    
      {
        "sensorType": "NH3",
        "value": 0.45,
        "unit": "PPM"
      }

Response Example:

    
      {
      "success": true,
        "data": {
          "id": 1,
          "sensorType": "NH3",
          "value": 0.45,
          "unit": "PPM",
          "createdAt": "2024-07-20T15:00:00.000Z",
          "updatedAt": "2024-07-20T15:00:00.000Z"
        }
      }

Get Average Data

URL: /v1/api/average

Method: GET

Query Parameters:

  • tanggal (required): The date in YYYY-MM-DD format

Request Example:

      GET /v1/api/average?tanggal=2024-07-20

Response Example:

      
        {
          "success": true,
          "data": {
            "NH3": {
              "average": "0.45",
              "unit": "PPM"
            },
            "PH": {
              "average": "7.10",
              "unit": "pH"
            }
          }
        }

Get Data With Day

URL: /v1/api/withday

Method: GET

Query Parameters:

  • tanggal (optional): The date in YYYY-MM-DD format. If not provided, defaults to today's date.
  • sensorType (optional): The type of sensor data to retrieve. Can be either NH3 or PH. Defaults to NH3.

Request Example:

    GET /v1/api/withday?tanggal=2024-07-20&sensorType=NH3

Response Example:

      
        {
        "success": true,
          "data": [
          {
            "id": 1,
            "sensorType": "NH3",
            "value": 0.5,
            "unit": "PPM",
            "createdAt": "2024-07-20 14:30:00"
          },
          {
            "id": 2,
            "sensorType": "NH3",
            "value": 0.3,
            "unit": "PPM",
            "createdAt": "2024-07-20 13:45:00"
          }
        ]
      }

Data PPM NH3 with Day

URL: /v1/api/rataratappmnhDays?tanggal=2024-07-20&sensorType=NH3

Method: GET

Query Parameters:

  • tanggal: (optional) The date in YYYY-MM-DD format. If not provided, the current date will be used.
  • sensorType: (optional) The type of sensor. Either 'NH3' or 'PH'. Default is 'NH3'.

Response Example:

 
        {
          "success": true,
          "data": [
            {
              "sensorType": "NH3",
              "averageValue": "0.45"
            }
          ]
        }
